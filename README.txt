 GLADYS version 1.0

 Gladys est un assistant domotique intelligent pour Raspberry Pi.

 PRE-REQUIS : ( installation automatique des prérequis avec le fichier install.sh)
 - Apache, PHP5, MySQL
 - NodeJS > 0.8
 - WiringPi et rcswitch-Pi
 - MOC
 - Les modules npm "xmlhttprequest" et "googlemaps"
 - L'utilisateur www-data ajouté au groupe audio


 INSTALLATION :
 les étapes sont décrites à cette adresse : http://intelligenceonline.synergize.co/developpeur

 Pour faire la MAJ de la 1.0 vers la 1.1, exécuter le script "maj.sh", puis relancer le code NodeJS pour faire la mise à jour de la base de donnée
